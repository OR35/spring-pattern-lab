# Spring Pattern Lab

Spring 기반 서버 개발 패턴을 정리한 개인 아카이브입니다.

기능 구현보다는 다음에 초점을 둡니다.
- 도메인 분리 기준
- 인증/인가 구조
- JPA 엔티티 설계 원칙
- API 계층 책임 분리

## Structure

- patterns/
  - 도메인, 인증, 영속성, API 설계 패턴
- examples/
  - 실제 프로젝트 기반 예시 코드
- dev-notes/
  - 시행착오, 리팩토링, 고민 기록

## Goal

- 서버 설계 시 반복되는 실수 제거
- Spring 기반 프로젝트 초기 구조 템플릿 확보
